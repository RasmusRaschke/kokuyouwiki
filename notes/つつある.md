# つつある

- Formation: V連用形+つつある
- Meaning: Expresses that some change is currently taking place gradually or about to be finished.
- Often used with gradual adverbs like 次第に・徐々に・だんだん・少しずつ

**Examples**
1. 政治に興味を持つ若者が次第に増えつつあるようだ。
2. 台風が九州接近しつつある。

- Never used with habitual actions
	- 図書館で勉強しつつある。-> wrong
	- 図書館で勉強している。-> correct


# つつ

- Formation: V1連用形+つつ+V2
- Meaning: Two actions V1 and V2 are done by *the same person* at the same time.

**Examples**
1. 彼は最後まで自分の学説が正しいことを主張しつつこの世界を去った。
2. 悪いことと知りつつ、友達に嘘をついてしまった。
