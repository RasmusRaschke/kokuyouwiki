# 黒曜ウィキ

Welcome to my personal Japanese grammar wiki. Here I collect Japanese grammar patterns and their meanings with examples. This is primarily for my own convenience, but I am happy if it is of some help for anybody else.
By the way, if you wonder about the name: I manage the wiki, like my MathWiki, in Obsidian, and I am very fond of 駄洒落. You might see where this is going...
